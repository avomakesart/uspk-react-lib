## Props

Accepts props for `HTMLAnchorElement` like href, etc. In addition

**as** `React.ElementType`

Links will be `<a>` by default. Pass this prop If you want it to be a different HTML element.

**secondary** `boolean`

Pass this prop to get the secondary style links.

**inverse** `boolean`

Pass this prop for links used over a dark color background.

**leftIcon** `Icon`

Accepts an Icon to display on the left

**className** `string`

Prop to pass an additional class to Link.

**ref** `React.Ref`

Forwards provided ref to underlying `a` element.


**onClick** `func`

If the `as` prop is passed as `button`, we will have an `onClick` prop, to execute a funciont


**buttonType** `string`

If the `as` prop is passed as `button`, we will have a `buttonType` prop, to specify what kind of button we will, like:
`button` | `submit` | `reset`
